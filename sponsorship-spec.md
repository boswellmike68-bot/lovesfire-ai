# LovesfireAI Sponsorship Specification

## Overview

This document outlines the sponsorship tiers, benefits, and payment structure for the LovesfireAI project.

## About LovesfireAI

LovesfireAI is a cinematic engine for AI-generated vertical video — converting long-form text scripts into stitched, 9:16, TikTok-ready MP4s with cinematic color grading, camera motion, and audio-reactive effects.

## Sponsorship Tiers

### Bronze Supporter ($25/month)
- Name listed in project README
- Access to sponsor-only updates
- Priority issue consideration

### Silver Sponsor ($100/month)
- All Bronze benefits
- Logo placement on project documentation
- Early access to new features
- Priority support for integration questions

### Gold Sponsor ($500/month)
- All Silver benefits
- Dedicated support channel
- Feature request priority
- Monthly consultation call
- Custom LUT preset development

### Platinum Partner ($1000+/month)
- All Gold benefits
- Custom pipeline integration support
- Co-marketing opportunities
- Direct access to maintainer
- On-premise deployment assistance

## Payment Methods

All sponsorship payments are processed securely through Stripe.
